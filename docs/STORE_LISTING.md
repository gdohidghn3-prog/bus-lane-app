## 자세한 설명 (최대 4000자)
```
서울 버스전용차로, 들어가도 되는지 매번 헷갈리지 않으세요?

「버스전용차로」는 현재 위치·시각·요일·공휴일·차량 종류를 종합해
지금 이 차선에 진입해도 되는지를 실시간으로 알려주는 운전 보조 앱입니다.

■ 핵심 기능

• 지도 위에 가까운 버스전용차로 구간을 색깔로 표시
  - 녹색: 진입 가능 (운영 시간 외 또는 예외 차종)
  - 빨강: 진입 금지 (단속 시간 운영 중)
  - 노랑: 곧 상태 변경 예정 (10분 이내 전환)

• 운전 중 백그라운드 경고
  - 단속 시간대에 버스전용차로 500m 이내 접근 시 알림
  - 200m 이내는 강한 경고 (실수로 진입하기 전에)

• 차량 종류별 예외 자동 적용
  - 일반 / 택시 / 9인승 이상 / 36인승 이상 / 응급
  - 차종을 한 번만 설정하면 매번 알아서 판단

• 정확한 시간 처리
  - 한국 표준시(KST) 기준
  - 평일·토요일·일요일·공휴일 자동 구분
  - 서울시 공식 운영 시간 반영

■ 이런 분께 추천

• 출퇴근·외근으로 서울 시내 주행이 잦은 운전자
• 택시·9인승 이상 다인승 차량 운전자 (예외 차종)
• 가끔 서울에 올라오시는 외부 지역 운전자
• 신규 면허 취득자

■ 개인정보 보호

• 회원가입 없음, 익명 사용
• GPS 좌표는 단말에서만 처리, 서버에 원본 미저장
• 경고 로그도 100m 격자로 익명화 저장
• 광고·마케팅·제3자 제공 없음

■ 중요 안내 (꼭 읽어주세요)

본 앱은 운전 보조 참고 서비스이며 법적 판단을 대체하지 않습니다.
GPS 오차·데이터 갱신 지연·임시 통제 등으로 실제 단속과 다를 수 있으므로
도로 표지판과 도로교통법을 항상 우선하여 판단해 주시기 바랍니다.
앱 안내를 신뢰하여 발생한 단속·과태료 등에 대해 본 앱은 책임지지 않습니다.
운전 중 앱 조작은 위험합니다. 정차 후 사용하세요.
```
