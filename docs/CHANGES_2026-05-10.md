# 2026-05-10 변경 요약 (복덩이 오케스트레이션)

## 결정
- 출시 전략: Android 우선, iOS 6개월 보류
- 월 운영비: $0 (Render Free + Supabase Free + GitHub Pages)
- 일회성 비용: $25 (Google Play Console)
- 백엔드 배포: Render
- 법률 문서 호스팅: GitHub Pages (`/docs`)
- RPC/PostGIS 최적화: 보류 (구간 100개 미만)

## 코드 변경
- `backend/src/services/segment-service.ts`: 읽기 쿼리를 anon 클라이언트로 전환, 경고 발생 시 `alert_logs` 비동기 INSERT (위치 100m 격자 익명화)
- `backend/migrations/004_rls_policies.sql`: RLS 정책 신규 (anon SELECT만 허용, alert_logs는 service key 전용)
- `backend/.env.example`: `NODE_ENV` 등 보강
- `mobile/.env.example`: 운영 URL/주석 보강
- `mobile/eas.json`: development/preview/production 프로파일 신규
- `mobile/app.json`: Android 권한 보강 (POST_NOTIFICATIONS, FOREGROUND_SERVICE_LOCATION), iOS UIBackgroundModes, expo-location/notifications 플러그인 설정
- `render.yaml`: Render Free 플랜 배포 청사진 신규
- `.github/workflows/ci.yml`: EAS 빌드를 `workflow_dispatch` 수동 트리거로 변경 (무료 크레딧 절약)
