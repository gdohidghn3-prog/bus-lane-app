# 버스전용차로 (Bus Lane Navigation)

서울 운전자가 "지금 이 차선 들어가도 되는가?"를 위치·시간·요일·공휴일·차종 기반으로 즉답받는 모바일 앱입니다.

## 법률 문서

- [개인정보처리방침](PRIVACY_POLICY)
- [이용약관](TERMS_OF_SERVICE)

## 면책

본 앱이 제공하는 정보는 참고용이며, **실제 도로 법규와 교통 표지판이 항상 우선합니다**. 앱의 안내와 실제 도로 상황이 다를 경우 도로 표지판·신호·경찰관의 지시를 따라 주세요.
